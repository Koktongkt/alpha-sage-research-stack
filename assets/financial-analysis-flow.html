Institutional Financial Analysis Flow

Discovery → evidence-led deep research → empirical validation → risk-adjusted portfolio decision → monitoring and learning

Discovery Is Two-Lane

  • • Classic value protects margin of safety
  • • Hidden-gem lane preserves upside asymmetry
  • • Both stop at a ranked shortlist

Deep Dive Builds the Thesis

  • • Shared evidence pack avoids duplicated research
  • • Specialists cover only material questions
  • • One final IC report after all reports return

Quantest Tests Claims

  • • Test 1–3 decision-relevant hypotheses
  • • Preserve point-in-time and final-holdout integrity
  • • Separate significance from investability

Capital Is Conditional

  • • No evidence means no forced position
  • • Size by conviction, liquidity, and downside
  • • Monitoring updates the thesis—not past data
